# Errata — *Continuous Intent Delivery*

Corrections, clarifications, and post-publication updates to *Continuous Intent Delivery* (Alchemaize, May 2026, ISBN 979-8-9925330-9-5).

A book this opinionated will be wrong about some things. This file tracks the corrections as they accumulate. Readers who find errors of fact, citation problems, or claims that no longer match the methodology as it has evolved are encouraged to file an issue against this repository. The authors review the issue queue monthly.

## How to read this file

Each entry names the chapter, page (where stable), and the correction. Entries are dated by the month the correction was confirmed. The most recent corrections appear at the top.

## 2026

### May 2026 — first printing

No errata yet recorded. This file will accumulate as readers report findings. The authors expect to populate it within the first few months of publication.

## Reporting an issue

The fastest way to get a correction into this file is to file an issue at https://github.com/alchemaize/cid/issues with the label `errata`. Please include:

1. The chapter and section (page numbers, where you have them).
2. The exact text as it appears in the book.
3. The correction, with a citation if the correction depends on an external source.

The authors review and either accept, reject, or escalate the finding within thirty days. Accepted findings appear in this file with attribution and (where the reader requests it) a credit line.

## Reader contributions

A second edition is planned for 2027 or 2028, depending on how much the practice changes in the next two years. Reader contributions to this file are the primary input to that revision. Substantial contributions are credited in the second edition's acknowledgments.
